# goals Folder Definition

`docs/goals/`은 제품의 목표와 의사결정 원칙을 담는 폴더다.

GCSE 역할: `Goal`

## Purpose

- V4가 어떤 제품 방향을 향하는지 정의한다.
- 개발 중 바뀌면 안 되는 판단 기준과 금지 원칙을 둔다.
- 세부 구현 상태나 체크리스트가 아니라 "왜 이 방향인가"를 설명한다.

## Allowed Documents

- 제품 목표 요약
- 핵심 설계 원칙
- 범위 밖으로 밀어낼 항목
- 장기 방향성과 우선순위 판단 기준

## Not Allowed

- 최신 진행률
- 구현 체크리스트
- 검증 로그
- 코드 파일 목록 중심의 설명

진행 상태가 필요하면 `docs/planning/`을 참조한다.
